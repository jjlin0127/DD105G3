楊桃甘

楊桃富含豐富的檸檬酸、草酸、蘋果酸等元素能有效的提高胃液的酸度，從而快速促進食物的消化已達到和健胃消食的作用。胸悶欲嘔、消化不良、腹脹打嗝，只要多吃楊桃，症狀便能逐漸減輕。

鳳梨甘

含有大部分人體中所需的維生素、礦物質，能有效幫助消化吸收，此外，鳳梨中的鳳梨酵素，能有效分解食物蛋白質，促進腸胃蠕動，幫助排便。

火龍果甘

火龍果有助於潤腸通便，還具有解除重金屬中毒、抗自由基等功效。

香蕉甘

有研究指出香蕉能有效降低腦部血管阻塞風險。研究回顧了從60年代開始的11個文獻後指出，每天三餐各吃一根香蕉，可有效降低腦部血管阻塞風險達21％。

草莓甘

草莓含果膠及抗氧化物較高，因此，可預防壞血病、動脈硬化和心血管可防止動脈硬化。

芭樂甘

芭樂中含有多酚，多酚中的類黃酮素，包含常聽到的兒茶素、槲皮素、前花青素、木犀草素等抗氧化元素。這些元素可以幫助清除血液中多餘的低密度膽固醇（壞膽固醇），預防血栓形成。

芒果甘

芒果富含維生素B6，它對神經系統和大腦的正常運作十分重要。芒果還含有谷氨醯胺，有助於提高人的專注度，並改善記憶。

蘋果甘

蘋果具有健腦的作用，能夠延緩大腦的衰老速度，並且能有效改善失眠的狀況，蘋果中含有的微量元素能起到緩解便秘，多吃蘋果可以健腦，對大腦有好處，同時有利於心臟的保健。
